### 프로젝트 소개

`냉장고를 Vue탁해 프로젝트`는 **사용자가 등록한 재료를 활용하여 맞춤 레시피를 제공하는 웹 블로그**입니다.

핵심 기능은 다음과 같습니다.

- 사용자가 자신의 냉장고 재료들을 등록하면 재료 데이터를 기반으로 맞춤 레시피를 제공합니다.
- 자신만의 요리 레시피 글을 작성할 때 `냉장고를 vue탁해`에서 제공하는 에디터를 통해 쉽고 즐겁게 등록할 수 있습니다.
